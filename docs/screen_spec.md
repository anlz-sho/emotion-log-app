# screen_spec.md

## このファイルについて

各画面の細かい動きを記載しています。

## Home画面

* 縦スクロール
* 感情グループごとにセクション分け
* ボタンは角丸
* 押しやすいサイズ
* 選択中は背景色変更
* 優しい雰囲気

## 保存内容
* 日時
* 選択感情
* メモ

## 保存仕様
* 1日に複数回保存できる
* 同じ日に2回目以降保存した場合、新しいログは作らない
* 既存の今日のログに感情を追加する
* 既存の感情は削除・上書きしない
* 重複する感情は追加しない
* メモは既存メモの末尾に追記する
* 追記時は時刻が分かるように区切る

例：
[09:20]
朝から焦った

[21:10]
夜は少し落ち着いた

## Home画面 削除仕様

* 今日の記録がある場合のみ「今日の記録を削除」ボタンを表示する
* 削除前に確認ダイアログを表示する
* 確認後、今日の EmotionLog を削除する

## Logs画面

保存済み感情ログを一覧表示する画面。

* Home, Calendar, Ranking と同じく TabView の1タブとして表示
* 保存済みログを新しい順で表示
* 日時を表示
* 選択した感情を表示
* メモを表示
* 縦スクロール可能

## Logs画面 UI方針

* シンプル
* 読みやすい余白
* カード風デザイン
* やさしい雰囲気

## Calendar画面

* 月表示カレンダーを表示
* 記録がある日は色付き表示
* 感情数に応じて色の濃さを変更
* 記録がない日は通常表示
* 今日の日付を分かりやすく表示
* 日付をタップ可能

## Calendar詳細表示

日付タップ時：
* その日の感情ログ一覧を表示
* 選択感情を表示
* メモを表示

## Calendar画面 UI方針

* シンプル
* やさしい色
* 一目で感情量が分かる
* カレンダーらしい余白

## Ranking画面

感情ログを集計し、期間や曜日ごとの傾向を確認する画面。

* 感情ランキングを表示
* 感情グループ別ランキングを表示
* 週別ランキングを表示
* 月別ランキングを表示
* 曜日別ランキングを表示
* 表示単位をタブまたは segmented control で切り替える
* 押された回数が多い順に並べる
* 回数を表示する
* スクロール可能

## Ranking画面 表示内容

### 感情ランキング

例：

* 不安：18回
* 焦る：15回
* ホッとした：12回

### グループランキング

例：

* 不安系：40回
* 落ち込み系：25回
* 良い感情系：20回

## Ranking画面 UI方針

* シンプル
* 見やすい余白
* 感情グループ色を活用
* やさしい雰囲気
