# app_spec.md

## このファイルについて

アプリ全体の目的・概要を記載しています。

## アプリの目的

嫌な日でも、
「全部が悪かったわけではない」
と振り返れるようにする。

感情の継続傾向を可視化し、
問題の長期化に気づけるようにする。

## アプリ概要

毎日の感情をボタンで記録し、
あとからカレンダーやランキングで振り返るiPhoneアプリ。

## MVP機能

* 感情ボタン入力
* メモ入力
* 保存
* ログ一覧表示
* カレンダー表示
* ランキング表示

## UI方針
- やさしい雰囲気
- 角丸ボタン
- 感情グループごとに色分け
- TabViewで画面切り替え

## 技術

* SwiftUI
* SwiftData
* iOS17以上
